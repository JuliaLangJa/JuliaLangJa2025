| 項目 | 内容 |
| --- | --- |
| 対象 | Julia言語に関連することであれば内容不問 |
| 日時 | 2025年12月13日(土) 13:00 - 19:00 |
| 方式 | 対面 / Zoom ハイブリット |
| 会場 | [東京科学大学 大岡山キャンパス レクチャーシアター](https://www.ssc.titech.ac.jp/amap/home/ookayama/west/lecture-bldg1/) |

| 参加登録 | 近日中に開始予定 |
